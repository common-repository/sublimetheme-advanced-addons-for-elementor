== Description ==

SublimeTheme - Advanced Adddons for Elementor is an Elementor Addons plugin which contains 7 advanced elementor widget that can create custom layout with ease.

7 Highly Customizable Widgets
1. Latest Posts
2. Icon Text
3. CTA (Call To Action)
4. Counter
5. Pricing
6. Team
7. WP Form (Contact Form)

== Installation ==

NOTE : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

1. Upload the plugin folder to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. You can type "sublime" on your element tabs within Elementor editor and all the available elements will appear.
4. Also you can see them under the category "SublimeTheme - Advanced Addons for Elementor" on your element/widget list.

== Frequently Asked Questions ==

= Can I use the plugin without Elementor Page Builder? =
No. You cannot use without Elementor since it's an addon for Elementor.

= Does it work with any theme? =
Absolutely! It will work with any theme where Elementor works.
